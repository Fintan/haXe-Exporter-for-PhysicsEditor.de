An exporter script for PhysicsEditor (www.physicseditor.de) to generate a haXe (.hx) class file.

Installation:

paste the folder called box2d-haXe into /Applications/PhysicsEditor.app/Contents/Resources/exporters/

Usage:

A demo can be found in the folder called 'eg'


The resulting .hx file is intended to be used with this port of box2d:
http://code.google.com/p/box2dhx/
